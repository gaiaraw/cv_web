EXPÉRIENCES

PROJET DE RECHERCHE ET DE DÉVELOPPEMENT - MASTER 1

Au cours de l’année de Master 1, avec mes collègues étudiants, j'ai participé à un projet de recherche et de développement portant sur l’analyse des données Oxford Nanopore provenant des cellules humaines infectées par le VIH. Ce projet, qui a duré trois mois, a principalement été axé sur l'utilisation de Python. Notre objectif était de mettre en place un pipeline pour traiter et analyser les données de nos clients, qui étaient des biologistes. Il était donc crucial que l'outil soit clair et simple d'utilisation pour des non-informaticiens.

Outils informatiques : Python, Nextflow (Groovy), IGV

Collaboration avec le MFP et le LaBRI.

DÉVELOPPEMENT WEB FULLSTACKS - MASTER 2

Pour mon stage de fin d'études, j'ai développé une application web destinée à des utilisateurs non experts de l’EEG et de l’informatique pour un usage clinique. L'application a été réalisée avec le framework Python Flask pour le backend et le système d'authentification. La communication avec la base de données s'effectuait via l'ORM SQLAlchemy. Le frontend a été élaboré avec Javascript, HTML5/CSS3 et Bootstrap. Pour l'analyse des signaux EEG, j'ai adapté du code Python écrit par mon co-encadrant et j'ai également rédigé sa documentation à l'aide de la bibliothèque Python Sphinx. La partie visualisation a été réalisée avec la bibliothèque de visualisation interactive Python Bokeh.

Outils informatiques : Python (Flask, SQLAlchemy, SQLite, Bokeh, Sphinx), HTML5/CSS3 (Bootstrap), Javascript

Projet réalisé au sein de l’équipe NCP du LIB de Sorbonne Université.
